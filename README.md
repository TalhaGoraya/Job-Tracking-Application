# Job Application Tracker

A full-stack job application tracker built with HTML, CSS, JavaScript, Node.js, Express, and SQLite.

## Features

- Track applications by company, role, location, status, and notes
- Live dashboard showing status counts
- CRUD operations via a REST API
- Persistent data in SQLite

## Run locally

1. Install dependencies:

   npm install

2. Start the server:

   npm start

3. Open `http://localhost:3000`
